Le SEO technique reste le fondement d'une stratégie de référencement réussie. En 2024, certains aspects sont devenus non-négociables.

## Core Web Vitals : toujours d'actualité

Google continue de valoriser l'expérience utilisateur à travers les Core Web Vitals :

- **LCP (Largest Contentful Paint)** : Moins de 2.5 secondes
- **FID/INP (Interaction to Next Paint)** : Moins de 200ms
- **CLS (Cumulative Layout Shift)** : Moins de 0.1

## Crawlabilité et indexation

### Le fichier robots.txt

Un robots.txt bien configuré est essentiel :

```
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /api/

Sitemap: https://example.com/sitemap.xml
```

### Sitemap XML

Votre sitemap doit :
- Être à jour automatiquement
- Ne contenir que les URLs canoniques
- Inclure les dates de modification

## Architecture de l'information

Une bonne architecture aide les moteurs de recherche à comprendre votre site :

1. **Hiérarchie claire** : Maximum 3 clics pour atteindre n'importe quelle page
2. **URLs descriptives** : `/blog/seo-technique-2024` plutôt que `/p?id=123`
3. **Maillage interne** : Liens contextuels entre contenus liés

## HTTPS et sécurité

En 2024, HTTPS est obligatoire. Mais allez plus loin :

- Headers de sécurité (CSP, HSTS)
- Certificat SSL valide et renouvelé
- Pas de contenu mixte (HTTP/HTTPS)
