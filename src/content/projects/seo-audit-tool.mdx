Un outil open-source pour automatiser les audits SEO techniques.

## Fonctionnalités

- Analyse automatique des Core Web Vitals
- Détection des problèmes de crawlabilité
- Vérification des balises meta et Open Graph
- Analyse du maillage interne
- Recommandations générées par IA

## Stack technique

- Node.js pour le backend
- Puppeteer pour le crawling
- Claude API pour les recommandations
- Vue.js pour l'interface

## Résultats

L'outil a été utilisé pour auditer plus de 100 sites, permettant d'identifier des améliorations qui ont conduit en moyenne à une augmentation de 30% du trafic organique.
